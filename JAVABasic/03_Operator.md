논리 부정 연산자 
논리 값을 부정하여 반대 값으로 변경
제어문을 활용할 때 많이 쓰임.

boolean boo1 = true;
boolean bool2 = !bool1;
System.out.println(bool2);

증감연산자.
1씩 더하거나 뺴는 연산자.
++  --

전위연산 : 먼저 연산 후 다른 연산 실행.
후위연산 : 다른 연산 우선 실행 후 연산.

 전위 연산자 예시
int a = 10;
int b = ++a;
System.out.println(a + ", " + b);

11, 11

후위 연산자 예시

int a = 10;
int b = a++;
System.out.println(a + ", " + b);
11, 10
